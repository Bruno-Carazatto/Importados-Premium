# 🚀 Importados Premium - Catálogo com WhatsApp

Landing page moderna para venda de produtos importados sob demanda, com integração direta via WhatsApp.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido com foco em negócios reais que trabalham com **encomendas personalizadas**, sem estoque fixo e sem sistema de pagamento online.

A solução permite que o cliente visualize os produtos e entre em contato diretamente via WhatsApp para realizar o pedido.

---

## 🎯 Objetivo

Criar uma interface moderna, rápida e responsiva que:

- Exiba produtos de forma organizada
- Facilite a comunicação com o cliente
- Transmita confiança e profissionalismo
- Funcione perfeitamente em dispositivos móveis

---

## 🧠 Conceito aplicado

- Catálogo digital inteligente
- Venda sob demanda
- Integração direta com WhatsApp
- UX voltada para conversão

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Responsivo)
- JavaScript (Vanilla)
- Bootstrap 5

---

## ⚙️ Funcionalidades

- 🔍 Busca de produtos em tempo real
- 🎯 Filtro por categorias
- 📱 Layout totalmente responsivo
- 💬 Botão de contato com mensagem automática no WhatsApp
- ✨ Animações suaves ao rolar a página
- 📦 Cards de produtos com layout premium
